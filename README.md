Sobre o Projeto

Este repositório contém o código do site Springlo Tech, criado para apresentar projetos autorais em Realidade Aumentada.

O objetivo do site é demonstrar aplicações práticas de AR em diferentes áreas:

Indústria (Manuais técnicos interativos)
Educação (Livros em Realidade Aumentada)
Varejo (Visualização de produtos em AR)
Tecnologias Utilizadas
HTML
CSS
JavaScript
WebAR
Model Viewer
Unity (projetos apresentados)
Blender (modelagem 3D)
